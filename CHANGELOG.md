## [1.0.0] - 2025-01-08

### Added

- Initial release
- 14 scene types: architecture, scaling, database, deployment, security, debugging, testing, performance, api, monitoring, frontend, success, error, default
- Auto-detection of scene type based on title keywords
- 4 themes: github-dark (default), dracula, nord, one-dark
- 30+ Lucide-style icons
- SVG primitive components: card, metric, status, arrow, codeSnippet, terminalBlock, titleBar
- Full TypeScript support with type definitions
- Comprehensive test suite (671 tests)
- Zero runtime dependencies
